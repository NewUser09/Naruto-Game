# 🍥 Naruto 2D Unity Game

Proyek ini adalah game 2D berbasis Unity yang terinspirasi dari dunia Naruto. Game ini menampilkan beberapa karakter ikonik dengan gaya gameplay sederhana namun seru.

## 🎮 Deskripsi

Game ini dibuat sebagai proyek pembelajaran dan eksplorasi dalam pengembangan game menggunakan Unity. Pemain dapat mengenal karakter-karakter populer dari dunia shinobi dan menikmati mekanisme permainan 2D.

## 🧑‍ ninja Karakter

Berikut beberapa karakter yang tersedia dalam game:

* Naruto Uzumaki
  Ninja dari Desa Konoha dengan mimpi menjadi Hokage.

* Sasuke Uchiha
  Rival Naruto dengan kekuatan Sharingan.

* Sakura Haruno
  Ninja medis dengan kekuatan fisik luar biasa.

* Kakashi Hatake
  Guru Tim 7 yang dikenal sebagai ninja peniru.

* Itachi Uchiha
  Ninja jenius dengan kemampuan genjutsu yang kuat.

* Jiraiya
  Salah satu Sannin legendaris dan mentor Naruto.

* Pain
  Pemimpin Akatsuki dengan kekuatan Rinnegan.

* Suigetsu Hozuki
  Ninja dengan kemampuan tubuh cair seperti air.

## ⚙️ Fitur

* Gameplay 2D sederhana
* Kontrol karakter dasar (gerak, lompat, attack)
* Sistem karakter dengan kemampuan unik
* Animasi sprite
* UI sederhana

## 🛠️ Teknologi

* Unity Engine
* C#
* Sprite 2D
* Animator Controller

## 🚀 Cara Menjalankan

1. Clone repository ini
2. Buka project di Unity Hub
3. Jalankan scene utama
4. Klik Play di Unity Editor

## 📌 Catatan

Project ini dibuat untuk tujuan pembelajaran dan tidak untuk tujuan komersial.
